1. Description

    Test I2C communication using I2C (master) and Temperature Sensor (ADT7420) located on the ADuCM302x EZ-Board

2. Processor

  • ADuCM302x

3. Tested with

  • ADuCM302x EZ-Board
  • IAR embedded Workbench
  • Segger J-Link Lite

4. Overview

    This example uses the I2C device driver to read the ADT7420 temperature sensor registers. The ADT7420 temperature sensor is located on the ADuCM302x EZ-Board.
    The device drivers can operate in one of three modes.
    • Blocking
    • Non-blocking
    • Callback
    This example demonstrates the I2C being operated in Non-Blocking mode.
    For more information on driver modes of operation please refer to the driver API documentation.

5. Hardware Setup

  1. ADuCM3029-EZ-Board configured with default settings with following jumper settings.

    • JP1 - Connect jumper between pins 3 and 5. Rest Open
    • JP2 - Connect jumper between pins 1 and 3. Rest Open
    • JP3 - Connect jumper between pins
    • JP4 - Connect jumper between pins
    • JP5 - Connect jumper between pins
    • JP6 - Connect jumper between pins
    • JP7 - Connect jumper between pins
    • JP8 - Not connected.
    • JP9 - Not connected.
    • JP10 - Connect jumper between pins 1 and 2. Rest open
    • JP11 - Connect jumper between pins 2 and 3.
    • JP12 - Not connected.
    • JP13 - Connect jumper between pins 1 and 2.

6. Running the Example

  1. Prepare hardware as explained in the Hardware Setup section.

  2. Build the project, load the executable to ADuCM302x, and run it

  3. Look for the debug information displayed on terminal IO.
